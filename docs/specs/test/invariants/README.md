# Tempo Invariants

## Stablecoin DEX
- **TEMPO-DEX1**: newly created order id matches next order id.
- **TEMPO-DEX2**: place ask / bid orders properly reflected in user and DEX balances.
- **TEMPO-DEX3**: cancel active order and withdraw funds properly reflected in user and DEX balances.
- **TEMPO-DEX4**: `amountOut >= minAmountOut` when swap exact amount in.
- **TEMPO-DEX5**: `amountIn <= maxAmountIn` when swap exact amount out.
